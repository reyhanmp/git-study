git add = buat masukin semua file kalian ke "staging enviroment", kasarnya sih buat nyiapin file kalian sebelum di commit
git commit -m "Pesan Kalian" = buat mem "upload" file yang sudah kalian add ke local repository
git push = buat mem "upload" file yang sudah kalian commit ke remote repository, di langkah ini seharusnya file kalian sudah muncul di github
